These are the results of build: $buildId

For general use in an IDE, you simply want to add the composite software site of
https://$DLPath/ide/$updateRelease
to get the latest updates from this "$updateRelease" stream.

Normally, the latest three builds are kept in the composite repositories, so that gives some ability to "revert" an update, as long as there has not been three published builds since it was updated.
